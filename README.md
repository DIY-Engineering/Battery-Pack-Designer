# Battery Pack Designer 
is a software written in python designed to help you build any kind of battery pack, from new or recycled lithium, NiMh, NiCd and even Lead Acid cells.
It combines three main modules: New cell battery calculator, Recycled cell battery calculator with capacity matching and an optional Delta limit (difference between the highest and lowest capacity groups)
and a Layout editor to help with cell placement (if you need the battery to have a non standard layout) and nickel strips connections.
The only requirement is that you need Python 3.12 installed on your system.
